# Writer's Personal Blog & Portfolio

A custom website built for a writer to host their personal blog and portfolio.

## ✨ Key Features

- **Type-Safe Content Management:** Utilizes **Velite** to transform MDX content into type-safe data.
- **Modern UI/UX:** Built with **Next.js** and **Tailwind CSS**, featuring a responsive design, dark/light mode theming, and smooth transitions.
- **Optimized Performance:** Server-side rendering (SSR) and static generation for fast page loads and optimal SEO.
- **Containerized Deployment:** Dockerized architecture with Nginx as a reverse proxy.
- **Custom Analytics:** Privacy focused analytics to track reader engagement.

## 🛠️ Tech Stack

- **Frontend:** TypeScript, Next.js, React, Tailwind CSS
- **Infrastructure:** Docker, Docker Compose, Nginx, PostgreSQL
